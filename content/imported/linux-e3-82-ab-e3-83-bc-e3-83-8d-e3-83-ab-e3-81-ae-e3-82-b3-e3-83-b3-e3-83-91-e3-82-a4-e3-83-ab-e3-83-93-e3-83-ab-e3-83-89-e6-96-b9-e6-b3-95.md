### 手順：(比較的新しい(2.6.18〜)を想定)

1.  Linuxカーネルをダウンロード
2.  .configファイルの作成
3.  make && sudo make modules\_install && sudo make install
4.  再起動。(再起動時のGRUBの選択画面で、インストールしたカーネルを選ぶ。)
